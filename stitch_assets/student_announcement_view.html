Midterm Project Guidelines

priority_high Important
calendar_today Posted Oct 28, 2023
person Prof. Alan Turing

Welcome to the midterm project phase of CS101. This assignment is designed to test your understanding of data structures and algorithmic complexity discussed in the first half of the semester.

Project Scope

You are required to build a simple inventory management system. The system must support basic CRUD operations (Create, Read, Update, Delete) and implement at least two distinct sorting algorithms to organize the inventory data. We expect clean, well-documented code that adheres to standard software engineering practices.

Key Deliverables

  • Source code hosted on your university GitHub repository.
  • A 2-page design document explaining your choice of data structures.
  • A short video presentation (max 5 minutes) demonstrating the system functionality.
info
